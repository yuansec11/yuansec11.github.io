# 1、设置背景图片以及透明度和圆角
将下好的图片放到根目录下\themes\next\source\images文件夹里。命名为background.jpg（名字可随意）  
然后打开根目录下\themes\next\source\css文件夹，用记事本打开main.styl文件，在后面添加代码  
<!--more-->
```
//设置背景图片
body {
  background: url(/images/background.jpg);  
  background-repeat: no-repeat;
  background-attachment: fixed;
  background-position: 50% 50%;
  color: var(--text-color);
  font-family: 'Lato', "PingFang SC", "Microsoft YaHei", sans-serif;
  font-size: 1em;
  line-height: 2;
}
```

# 2、添加侧边栏链接图标
去阿里矢量图官网下载图标，svg格式的。
将下好的图片放到根目录下\themes\next\source\images文件夹里。
然后打开根目录下\themes\next\source\css文件夹，用记事本打开main.styl文件，在后面添加代码
```
.bilibili {
  background-image: url('/images/bilibili.svg');
  background-size: 1em 1em;
  opacity: 0.55;
  background-position: 0.05rem 0.2rem;
  background-repeat: no-repeat;
  height: 1rem;
  width: 1rem; 
  border-radius: 0rem;
  /*鼠标停留在图标上时，图标呈现发光效果*/
  &:hover {
      opacity: 1;
    }
}
```
最后用记事本打开根目录下\themes\next文件加下的_config.yml文件。
找到social项，仿照其他格式添加名字、链接、图标

# 6、添加小人
首先安装live2d模块  
```
npm install --save hexo-helper-live2d
```

然后安装你想要的人物模型,比如  
```
npm install live2d-widget-model-wanko
```

最后在根目录下的文件中添加代码   
```
live2d:
  enable: true
  scriptFrom: local
  pluginRootPath: live2dw/
  pluginJsPath: lib/
  pluginModelPath: assets/
  tagMode: false
  log: false
  model:
    use: live2d-widget-model-wanko//你想要的模型
  display:
    position: right
    width: 300
    height: 450
  mobile:
    show: true
```
